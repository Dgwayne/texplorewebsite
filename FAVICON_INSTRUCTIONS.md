# Favicon Setup

## Where to Save Your Favicon

For Next.js 13+ (App Router), save your favicon in the `/app/` directory:

**Save as:** `/app/favicon.ico`

## File Format

- **Preferred:** `.ico` file (favicon.ico)
- **Alternative:** `.png` file named `icon.png` (Next.js will auto-detect)

## File Location

```
texplore website/
└── app/
    └── favicon.ico  ← Save your favicon here
```

## Steps

1. Take your favicon image file
2. If it's a PNG, you can:
   - Convert it to ICO format (use an online converter), OR
   - Save it as `icon.png` in the `/app/` folder
3. Save it to: `C:\temp\texplore website\app\favicon.ico`

## Size Recommendations

- **favicon.ico:** 16x16, 32x32, or 48x48 pixels
- **icon.png:** 32x32 or larger (Next.js will optimize)

Once saved, Next.js will automatically use it as your site's favicon!
